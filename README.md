Web UI Components Project

This project contains three simple web interface designs created using HTML and CSS.
Each part demonstrates different UI layouts and design components.

1. Product Cards UI

Description

The **Product Cards UI** displays products in card format with product name, price, and stock status.

Features

* Three product cards displayed using **Flexbox**
* Product name and price
* Stock status indicator (In Stock / Out of Stock)
* Clean card layout with rounded corners and shadow

Products Displayed

* Wireless Headphones — $129.99 (In Stock)
* Mechanical Keyboard — $89.99 (Out of Stock)
* Smart Watch — $199.99 (In Stock)

Technologies Used

* HTML
* CSS
* Flexbox Layout

2. Library Management System UI

Description

This interface represents a **simple library management system layout** where users can search books and add new ones.

Features

* Search bar to search books
* Input fields for book title and author
* Add Book button
* List of books displayed in card format
* Remove button for each book

Example Books

* The Great Gatsby — by S. Scott Fitzgerald
* To Kill a Mockingbird — by Harper Lee
* The Great Gatsby — by F. Scott Fitzgerald

Technologies Used

* HTML
* CSS
* Responsive layout design.
